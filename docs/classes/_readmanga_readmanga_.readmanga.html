filterUpdatedManga

  • filterUpdatedManga(mangaUpdatesFoundCallback: (updates: MangaUpdates) => void, time: Date, ids: string[]): Promise<void>

  • Inherited from ReadManga.filterUpdatedManga

    • Defined in node_modules/paperback-extensions-common/dist/base/Source.d.ts:97

    (OPTIONAL METHOD) A function which should scan through the latest updates section of a website, and report back with a list of IDs which have been updated BEFORE the supplied timeframe. This function may have to scan through multiple pages in order to discover the full list of updated manga. Because of this, each batch of IDs should be returned with the mangaUpdatesFoundCallback. The IDs which have been reported for one page, should not be reported again on another page, unless the relevent ID has been detected again. You do not want to persist this internal list between {@link Request} calls

    Parameters

    • mangaUpdatesFoundCallback: (updates: MangaUpdates) => void

      A callback which is used to report a list of manga IDs back to the API

        • (updates: MangaUpdates): void

        • Parameters

          • updates: MangaUpdates

          Returns void

    • time: Date

      This function should find all manga which has been updated between the current time, and this parameter's reported time. After this time has been passed, the system should stop parsing and return

    • ids: string[]

    Returns Promise<void>
